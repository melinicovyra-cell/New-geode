# Greeting Mod для Geometry Dash

Этот мод добавляет приветственное сообщение при входе в главное меню Geometry Dash. Он создан с использованием Geode SDK и настроен для автоматической компиляции через GitHub Actions.

## Функционал
- При каждом запуске главного меню (`MenuLayer::init`) появляется всплывающее окно (`FLAlertLayer`) с приветствием.

## Требования
- **Geometry Dash:** версия 2.2081
- **Geode SDK:** версия 5.6.1 или выше

## Как получить мод (.geode)
Поскольку этот репозиторий настроен на GitHub Actions, вы можете скачать скомпилированный файл мода прямо из GitHub:
1. Перейдите на вкладку **Actions** в верхней части страницы репозитория.
2. Выберите последний успешный запуск workflow (обычно называется "Build Geode Mod").
3. Прокрутите вниз до раздела **Artifacts**.
4. Скачайте архив `geode-mod`.
5. Распакуйте его и скопируйте файл `.geode` в папку `geode/mods` вашей игры.

## Разработка
Для локальной сборки вам потребуется установленный Geode SDK. Используйте CMake для генерации проекта.
